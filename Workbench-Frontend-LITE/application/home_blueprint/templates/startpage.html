Welcome to the Analytics Workbench!

This Workbench offers tools to analyze Citizen Science Projects. As of now it utilizes Explicit Semantic Analysis (ESA) to assign Research Areas and SDGs to the Projects and Named Entity Recognition (NER) to identify Named Entities in the descriptions.

This is the "LITE" Version of the Workbench created within CS Track. The Workbench LITE can be used to analyze and evaluate project description. The full version also holds a database to save project names, urls, descriptions and their analysis results. From the saved data it offers further insights into connections between projects. If you are interested in this full version please contact us and we can discuss what is possible.

To analyse a project description you may enter it at the bottom of this page or go to the "Analyse Project" Page and enter it there. You can then use the button "Analyse Project" to start the analysis process.
