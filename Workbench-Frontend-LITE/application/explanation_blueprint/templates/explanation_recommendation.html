How do the recommendations work?

To calculate recommendations we use a personalized PageRank. PageRank (Page, Brin, Motwani & Winograd, 1998) was introduced in 1998 and Google was developed to test the algorithm. It is mainly used to create rankings in networks of websites with the links leading from one website to another serving as directed edges.

The basic idea of the PageRank Algorithm is explained as a Random Surfer that surfs through the network. This Surfer randomly moves from one Website to another using the links. Occasionally he will get bored and jump to any website regardless of links. The PageRank of a website represents the probability of the Random Surfer visiting that website if he moves through the network indefinitely.

The algorithm can be personalized by defining which websites the Random Surfer cam randomly jump to when not following links.
